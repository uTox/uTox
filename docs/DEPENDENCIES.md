# Dependencies

Before trying to compile uTox make sure you have all of the required dependencies.

## Operating Systems

- [Linux](#Linux)
- [FreeBSD](#FreeBsd)
- [OpenBSD](#OpenBSD)
- [Windows](#Windows)
- [OSX](#OSX)

## Linux

| Name         | Required              |
|--------------|-----------------------|
| cmake >= 3.2 | yes                   |
| check        | no (needed for tests) |
| dbus         | no                    |
| filter_audio | no                    |
| freetype     | yes                   |
| GTK          | no (runtime only)     |
| opus         | yes                   |
| libvpx       | yes                   |
| openal       | yes                   |
| toxcore      | yes                   |
| v4l          | yes                   |
| xext         | yes                   |
| xrender      | yes                   |
